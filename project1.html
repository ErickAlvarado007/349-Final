Friend Finder App

This is a simple friend finder app that helps you find friends based on your interests and proximity.
I worked on this project with my group members for my 491 Senior Capstone project.
we made it as a mobile app using React Native and Expo.
The app allows users to create a profile, set their preferences, and find potential friends nearby.
The app also includes a chat feature that allows users to communicate with each other.
The app is designed to be user-friendly and easy to navigate.
The app is built using React Native and Expo, which allows for cross-platform compatibility.
